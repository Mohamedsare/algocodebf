# Dossier des fichiers de tutoriels

Ce dossier contient tous les fichiers uploadés dans les tutoriels.

## Types de fichiers acceptés

### 🎬 Vidéos (Max 50 MB)
- MP4 (recommandé)
- WebM
- AVI
- MOV (QuickTime)
- WMV
- MPEG

### 📁 Autres fichiers (Max 5 MB)
- PDF
- Images (JPG, PNG, GIF, WebP)
- Documents (DOC, DOCX)
- Archives (ZIP)
- Code source (TXT, PY, JS, HTML, CSS, etc.)

## Configuration requise

Pour uploader des vidéos, assurez-vous que PHP est configuré avec :
- `upload_max_filesize` = 51M
- `post_max_size` = 52M
- `max_execution_time` = 300

Consultez `CONFIGURATION_VIDEOS.md` pour plus de détails.
