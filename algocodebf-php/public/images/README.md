# Dossier Images

Ce dossier contient les images statiques de l'application AlgoCodeBF.

## Structure recommandée

```
images/
├── logo.png              # Logo principal de AlgoCodeBF
├── logo-white.png        # Logo blanc (pour fond sombre)
├── favicon.png           # Favicon du site
├── default-avatar.png    # Avatar par défaut
├── hero-bg.jpg          # Image de fond hero section
└── placeholders/        # Images placeholder
```

## Notes

- Utiliser des images optimisées pour le web (compression)
- Format PNG pour les logos avec transparence
- Format WEBP ou JPG pour les photos
- Tailles recommandées :
  - Logo : 200x60px
  - Avatar par défaut : 200x200px
  - Favicon : 32x32px
  - Hero : 1920x1080px
